  2.4.1 Stream Initiate Request

   It carries a "Max Buffer Size" value (2 bytes), a “Type Included” boolean flag (1 bit in a byte; see
   below for meaning) and a "Source Stream ID" (1 byte) in the data section. The combination of source,
   destination, and Source Stream ID must uniquely identify this stream transmission.
